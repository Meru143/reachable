# Code of Conduct

This project follows a Contributor Covenant 2.1 style code of conduct.

## Our Standards

Examples of behavior that help build a positive community:

- Showing empathy and patience in discussions
- Giving and accepting constructive feedback
- Focusing on what is best for the project and its users
- Respecting different viewpoints, experience levels, and backgrounds

Examples of unacceptable behavior:

- Harassment, intimidation, or discriminatory language
- Personal attacks, insults, or trolling
- Publishing private information without permission
- Disruptive or bad-faith participation in issues, reviews, or discussions

## Scope

This code of conduct applies in project spaces and in public spaces when someone is representing the project.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this code of conduct.

Repeated or severe violations may lead to temporary or permanent bans from project spaces.

## Reporting

If you experience or witness unacceptable behavior, report it privately through the contact path in [SECURITY.md](./SECURITY.md). Maintainers will review reports promptly and handle them with discretion.
